SimpleTimesheet

Track your hours. Build timesheets. Email approvers. One click.

Features Platforms Install GitHub

SimpleTimesheet is a cross-platform time tracking app. Store data in your own cloud folder (iCloud, Google Drive, OneDrive) so it syncs across devices. No account, no analytics, no cloud lock-in.

Features

  • Quick time tracking – Start and stop the clock with one tap or click.
  • Work descriptions – Add notes when you stop the clock.
  • Timesheet generation – Weekly, bi-weekly, or monthly timesheets.
  • Email integration – Draft emails with embedded timesheet or CSV attachment.
  • Cross-device sync – One folder (iCloud/Google Drive/OneDrive) shared across devices.
  • Custom templates – Email subject and body templates with placeholders.

Platforms

macOS

Menu bar app, keyboard shortcuts (⌘⇧S start/stop, ⌘⇧T timesheet), settings window, notifications.

iOS

Timer, History, Timesheet, Settings tabs. Home screen widgets. App Shortcuts. Stop-from-widget opens app to add description.

Android

In progress via Skip. Same data format for cloud sync.

Install

Download the latest release (macOS app ZIP, unsigned iOS IPA for sideloading).

Or clone and build from source in Xcode.
